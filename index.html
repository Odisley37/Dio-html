1. Tags de Estrutura

(voltar)

HTML (HyperText Markup Language) é a linguagem de marcação padrão para documentos projetados para serem exibidos em um navegador da web. Ele pode ser assistido por tecnologias como Cascading Style Sheets (CSS) e linguagens de script como JavaScript.

HTML, ou HyperText Markup Language, é a linguagem de marcação padrão para a criação de páginas da web. Ela é usada para descrever a estrutura e o conteúdo de uma página da web, e é interpretada por navegadores da web para exibir a página para os usuários.

O HTML é composto por tags, que são conjuntos de caracteres que definem o significado e a estrutura do conteúdo de uma página da web. Por exemplo, a tag

define um parágrafo, a tag define uma imagem, e a tag define um link.

O HTML é uma linguagem relativamente simples de aprender, e é uma ótima maneira de começar a criar páginas da web. Existem muitos recursos disponíveis para ajudar os iniciantes a aprender HTML, incluindo tutoriais online, livros e cursos.

Os elementos básicos do HTML são os seguintes:

Tags de abertura e fechamento: as tags de abertura e fechamento são usadas para definir o início e o fim de um elemento HTML. Por exemplo, a tag p é usada para definir um parágrafo, e a tag p é usada para definir o fim do parágrafo. Atributos: os atributos são usados para fornecer informações adicionais sobre um elemento HTML. Por exemplo, o atributo alt é usado para fornecer um texto alternativo para uma imagem. Conteúdo: o conteúdo é o texto ou os elementos HTML que são exibidos na página da web.

As tags de estrutura definem a estrutura do documento HTML. As principais tags de estrutura incluem:

!DOCTYPE html: Define o tipo de documento e a versão do HTML.

html: Define o início de um documento HTML.

head: Contém metadados/informações sobre o documento.

body: Contém o conteúdo visível da página da web.

Alguns exemplos de elementos HTML

Aqui estão alguns exemplos de elementos HTML comuns:

Parágrafos: os parágrafos são usados para agrupar texto em blocos. Imagens: as imagens são usadas para adicionar imagens a uma página da web. Links: os links são usados para criar hiperlinks para outras páginas da web. Listas: as listas são usadas para organizar texto ou imagens. Formulários: os formulários são usados para coletar informações dos usuários.

Tags de Texto

(voltar)

As tags de texto são usadas para formatar o texto. Algumas das principais tags de texto incluem:

  • h1 a h6: Define títulos de diferentes tamanhos.
  • p : Define um parágrafo.
  • strong : Define texto em negrito.
  • em : Define texto em itálico.

Tags de Imagem

(voltar)

A tag de imagem é usada para incorporar imagens no documento HTML.

img: Define uma imagem. Os atributos mais comuns usados com a tag img são src (define o caminho da imagem) e alt (define o texto alternativo que deve ser exibido se a imagem não puder ser exibida)

Tags de Link

(voltar)

As tags de link são usadas para criar links.

a: Define um hiperlink. O atributo mais comum usado com a tag a é href (define o destino do link).
